### 1. 개발 환경 및 설정 소개

이 게임은 Windows 11 x64 환경에서 Visual Studio 2022 Community로 개발 및 테스트를 진행하였습니다.\
SDL 2.28.5 버전에서 동작하도록 구성했습니다.
